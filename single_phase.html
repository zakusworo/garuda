Single-Phase Flow (garuda.physics.single_phase)

Physics module - Governing equations for reservoir flow.

Single-phase and multiphase flow implementations.

class garuda.physics.single_phase.SinglePhaseFlow[source]

Bases: object

Single-phase flow in porous media.

Governing equation:

∂(φρ)/∂t + ∇·(ρu) = q

where u = -(k/μ)·(∇p - ρg∇z) (Darcy’s law)

Mass per unit volume at the current state: φ·ρ.

Returned by itself (not divided by dt). The time-derivative ∂(φρ)/∂t is formed by the caller as (accumulation_new − accumulation_old) / dt; see step_implicit().

compute_flux(solver)[source]

Compute net mass outflow per cell from TPFA face fluxes.

Sign convention: cell_flux[i] = Σ_f q_f summed with the outward orientation, i.e. positive value means net mass leaving the cell. This is consistent with the residual

V * ∂(φρ)/∂t + cell_flux − source = 0

used in step_implicit() and works in 1D, 2D, and 3D via the shared face_cells map.
