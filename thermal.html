Thermal Flow (garuda.physics.thermal)

Thermal flow module - Non-isothermal reservoir simulation.

Essential for geothermal applications where temperature affects: - Fluid properties (density, viscosity) - Rock properties (thermal expansion) - Heat transport (conduction + convection)

class garuda.physics.thermal.ThermalFlow[source]

Bases: object

Non-isothermal flow in porous media.

Coupled equations:

Mass: ∂(φρ)/∂t + ∇·(ρu) = q Energy: ∂(ρCpT)/∂t + ∇·(ρCpTu) - ∇·(λ∇T) = Q_T

where u = -(k/μ(T))·(∇p - ρ(T)g∇z)
