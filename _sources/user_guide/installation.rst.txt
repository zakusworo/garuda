Installation
============

Requirements
------------

- Python 3.9 or higher
- NumPy, SciPy
- Optional: Numba (for 3D solver acceleration)

From Source
-----------

.. code-block:: bash

   git clone https://github.com/zakusworo/garuda.git
   cd garuda
   pip install -e ".[dev,docs]"

Development dependencies include pytest, black, ruff, mypy, Sphinx, and ReadTheDocs theme.
